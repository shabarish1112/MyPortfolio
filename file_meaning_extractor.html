File Metadata Extractor

Metadata Parsing Tool

Python Flask String Manipulation

Overview

File Metadata Extractor is a utility tool that automates the extraction of meaningful metadata, such as creation dates and times, directly from filenames. It is particularly useful for organizing file archives where metadata might be lost but preserved in the naming convention (e.g., WhatsApp images or camera photos).

Key Features

  • Automated Parsing: Instantly extracts date and time components from standard filename formats.
  • Format Support: Designed to handle formats like IMG_YYYYMMDD_HHMMSS.jpg.
  • Data Reformatting: Converts raw string data into human-readable Date (DD-MM-YYYY) and Time (HH:MM:SS) formats.
  • Error Handling: Validates filename patterns to ensure accurate extraction.

Technologies Used

Built using Python with Flask for the web interface, demonstrating backend string processing and logical validation.

from flask import Flask, render_template, request

app = Flask(__name__)

@app.route("/", methods=["GET", "POST"])
def index():
    result = ""

    if request.method == "POST":
        filename = request.form["filename"]

        # Example: IMG_20240118_193012.jpg
        name_part = filename.split(".")[0]
        
        parts = name_part.split("_")

        if len(parts) >= 3:
            date = parts[1]
            time = parts[2]

            year = date[0:4]
            month = date[4:6]
            day = date[6:8]

            hour = time[0:2]
            minute = time[2:4]
            second = time[4:6]
