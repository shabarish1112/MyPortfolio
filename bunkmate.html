Bunkmate

Smart Attendance Tracking Tool

Python HTML/CSS Web Development

Overview

Bunkmate is a simple yet effective web tool designed to help students manage their attendance. It calculates current attendance percentages and provides instant visual feedback‚Äîgreen for safe, red for danger‚Äîbased on a user-defined threshold.

Key Features

  • Attendance Calculator: Instantly computes percentage based on classes attended vs. total classes.
  • Visual Alerts: Shows a Green indicator if you are safe, or Red if you are running low on attendance.
  • Custom Limits: Users can set their own "safe" percentage threshold (e.g., 75% or 85%).
  • Responsive Interface: Works smoothly on both desktop and mobile devices.

Technologies Used

Built using Python for backend logic (if applicable) and standard HTML/CSS for a clean, user-friendly frontend interface.
